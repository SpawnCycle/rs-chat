# Chat usage

## General

In case of popup you can always exit them by pressing escape.
If you see a popup with multiple fields, you can switch between the two with Tab.
On most screens you can exit them by pressing Ctrl+q.

## Chat

Ctrl+n: rename yourself in the current room
Ctrl+h: view this help popup
Ctrl+l: view the logs
Ctrl+f: view everyone in the current room
Ctrl+r: open the room join modal
Ctrl+s: open the room switch modal
Ctrl+b: toggle the side bar
Ctrl+d: scroll down
Ctrl+u: scroll up
Ctrl+p: toggle the offset mode (possible modes: relative, absolute)
Ctrl+t: show the notifications
Alt+p: disable the offset
Alt+d: leave the current room

## Logs

ArrowUp: go up
ArrowDown: go down
ArrowLeft: decrease the shown log level for the selected target
ArrowRight: increase the shown log level for the selected target
PageUp: scroll up
PageDown: scroll down
f: focus on the selected log target
h: hide the side bar
